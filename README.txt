== Description ==

**Shortcodes and Widgets for Share Theme Plugin**

This plugin is based on the well-known WordPress default widget 'Recent Posts' and enhanced to display thumbnails of the posts.

**Widget List:**
Recent Post With Thumbnail
About me

Add sexy-buttons, beautiful  dropcaps, column layouts and much more to WordPress. WP Shortcode makes it happen with a click of a button.

**Shortcodes Included:**
Dropcap (two styles - unlimited colors), bulleted list, columns layout, icons (Icons from Fontawesome), Sexy Buttons, Progress Bar.

== Installation ==
Installation Instructions

1.  Upload the whole plugin directory to the /wp-content/plugins/ directory
2.  Activate the plugin through the 'Plugins' menu in WordPress
3.  Enjoy !

== Frequently Asked Questions ==
Shortcodes + button is not working 

Please disable all plugins and check if shortcode plugin is working properly. Then you can enable all plugins one by one to find out which plugin is conflicting with WP Shortcode plugin.
